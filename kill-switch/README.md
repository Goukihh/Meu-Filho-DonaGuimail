# Kill Switch Server

Servidor de controle remoto para Meu Filho DonaGuimail.

## Como usar:

1. Instalar dependências:
```bash
npm install
```

2. Executar servidor:
```bash
npm start
```

3. Acessar painel: http://localhost:3000

## Deploy no Railway:

1. Conectar repositório no Railway
2. Configurar pasta: kill-switch
3. Deploy automático
